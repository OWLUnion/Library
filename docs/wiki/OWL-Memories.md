# OWL Memories

## 什么是OWL Memories

OWL Memories 是 OWL Server 旧周目的存档。在 OWL Server 6th. 及之后，OWL Memories 还支持下载整个服务端。每一个周目结束后，我们都会将存档上传至云盘或 GitHub 以供下载。
