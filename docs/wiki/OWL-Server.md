# OWL Server

## 什么是 OWL Server

OWL Server 是一个以“自由、开放、平等、互爱”为理念的 Minecraft 社区服务器。当前运行周目为 **OWL Server X4**，定位为互通生存服，面向 Java 与基岩玩家开放。

X4 主打养老生存、主城共建、轻经济和长期社区建设。服务器不会出售 OP、创造、神装、常驻飞行等破坏平衡的权限，金币主要用于便利、建筑辅助、领地、公寓、传送和装饰。

## 标语

Our Wild Land, Our Wonderful Love.

自由，开放，平等，互爱。

## 如何进入 OWL Server

你可以在 [加入服务器](/play/) 页面查看当前地址并导入游戏。

```text
当前周目：OWL Server X4
IP 地址：kupars.top
基岩版端口：19132
```

请优先阅读 [OWL Server X4](/wiki/OWL-Server-X4.html)。该页面会记录当前周目的定位、特色和最新说明。

## 遇到问题怎么办

如果你无法进入服务器，可以先检查以下事项：

- 客户端版本是否与服务器兼容。
- 网络是否能正常连接 `kupars.top`。
- 基岩版端口是否填写为 `19132`。
- 是否正在维护或临时重启。

如果问题仍然存在，请在 OWL 社区频道中反馈，并尽量附上截图、错误提示和你的客户端版本。

## 发现玩家作弊怎么办

请在 OWL 社区频道中反馈，或私聊管理组。反馈时尽量提供截图、录像、时间、坐标和玩家名，方便管理组核查。
