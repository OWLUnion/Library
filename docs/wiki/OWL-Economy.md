# OWL 经济系统

OWL 的金币系统用于维持服务器内的便利服务、领地、公寓、传送和装饰消费。

:::

## 获取方式

| 方式 | 数额 |
|---|---:|
| 现金购买 | 1 元 = 1000 金币 |
| 每日签到 | 500 金币/天 |
| 每日任务 | 500 金币/天 |
| 在线奖励 | 每 30 分钟 50-150 金币，最高600金币/天 |
| 活动奖励 | 参考具体活动规则 |
| 在线奖励上限 | 600 金币/天 |

## 用途

| 项目 | 价格 |
|---|---:|
| 飞行券 | 3000 金币 / 30 分钟（使用后不间断计时） |
| 第 1 块额外领地 | 3000 金币 |
| 第 2 块额外领地 | 5000 金币 |
| 第 3 块额外领地 | 8000 金币 |
| 第 4 块起 | 10000 金币/块 |
| 租用主城公寓（拥有配套服务） | 3000 金币 / 7 天 |
| 购买建筑（部分拥有配套服务） | 参考出售者制定的价格 |
| 补给包 | 3000 金币 | 食物、火把、基础工具、少量经验、建筑材料 |
| 加强包 | 15000 金币 | 飞行券、额外领地额度、进阶物品 |
| /rtp 传送 | 150 金币 |
| /back 传送 | 100 金币 |
| 前往下界 | 100 金币 |
| 前往末地 | 200 金币 |

::: warning 调整说明

经济系统会根据服务器活跃度、金币流通和玩家反馈继续调整。游戏内实际配置与本页面不一致时，请以游戏内公告和管理组说明为准。

:::
