## OWL 基金会

OWL 基金会负责管理玩家赞助。自 2022 年 11 月 OWL 管理组开启赞助渠道以来，基金会陆续收到多位玩家的支持，我们对此深表感谢。除记入基金会的赞助外，许多玩家也通过服务器、活动奖金等方式提供了间接支持，详情见下方“基金会之外的资金支持”。玩家赞助资金将全部用于 OWL Server 建设与维护。以下为最新赞助记录和资金使用情况；未知姓名的玩家已用收款页面显示的不完整名称代替。如名称或款项有误，请联系我们更正。

# 声明

此页面所述内容均为真实记录。玩家赞助记录将被保留并可供查询。OWL Server 是公益服务器，不设置任何强制赞助，也不会因赞助与否区别对待玩家。本页面旨在保持 OWL 基金会的公开、透明与可追溯。

# 财产归属

与 OWL Server 相关的资金和财产主要包括 OWL 基金会资金，以及基金会之外的资金或资源支持。
OWL 基金会资金由 OWL 安全中心全体成员共同管理。用于购买服务器、续费域名等服务器运营事务时，OWL 安全中心成员可在合理范围内使用基金会资金。
基金会之外的资源支持中，玩家提供的服务器仍归提供者所有；OWL 安全中心仅在提供者授权范围内使用相关服务器资源。
