# 财产归属

与 OWL Server 相关的资金财产涉及 OWL 基金会资金和基金会之外的资金支持两个方面。
OWL 基金会的资金由 OWL 安全中心全体成员共同拥有和支配。OWL 安全中心的全体成员在购买服务器及域名等服务器运营方面对 OWL 基金会资金有合理的使用权。
基金会之外的资金支持中，玩家提供的服务器属于玩家所有，OWL 安全中心对玩家服务器的使用权由玩家给予。
