# 即刻加入 OWL Server

OWL Server X4 是服务器当前运行中的周目。Java 版玩家可以通过 `kupars.top:35565` 加入；基岩版玩家可以通过 `kupars.top:39132` 加入。

这里是 OWL Server 的公开知识库，用于整理加入方式、服务器现状、经济系统、公告、资金明细和历史资料。旧周目的记录会继续保留，新玩家请优先阅读 X4 相关页面。
